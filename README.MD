Based on 2016 paper:

## ["UNSUPERVISED REPRESENTATION LEARNING WITH DEEP CONVOLUTIONAL GENERATIVE ADVERSARIAL NETWORKS"](https://github.com/llealgt/DCGAN_SVHN/blob/master/1511.06434.pdf)
